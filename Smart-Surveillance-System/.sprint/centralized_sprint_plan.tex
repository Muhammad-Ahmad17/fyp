\section{Risk Mitigation (5-Month Compression)}
%===============================================================================

\begin{itemize}
    \item \textbf{Parallel work:} B trains CV while A builds platform --- do not wait for Jetson until Sprint 4.
    \item \textbf{Simulator:} Member A's event simulator unblocks all backend sprints without B hardware.
    \item \textbf{Scope freeze:} 6 detection classes only; no goggles/forklift.
    \item \textbf{Slip buffer:} Optional gloves/boots dropped if Sprint 3 training behind schedule.
    \item \textbf{Integration weeks:} Non-negotiable --- missing two in a row triggers supervisor escalation.
\end{itemize}
